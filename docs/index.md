# TRON Documentation

TRON is a lightweight AI compute orchestrator for distributed Python workloads. This documentation bundle provides quick-start guides, examples, and troubleshooting for developers.

- Install: `pip install tron-client-py`
- Quickstart: see ../QUICKSTART.md
- Examples: see ../examples/

## What people search for

TRON helps developers find answers to distributed computing problems such as:

- "distributed Python compute"
- "self-hosted AI inference"
- "task orchestration for Python"
- "remote function execution"
- "model scoring and batch inference"

## Why TRON appears for your problem

TRON is designed for teams that want a lightweight, self-hosted alternative to Ray and Celery with transparent remote execution and a Python-native developer experience. Use `tron-client-py` for search-friendly package install and documentation.
